When making changes to any of the following files, remember to update the corresponding section in `README.md`:

| File | README Section |
|------|----------------|
| `index.html` | About, Sections (Home) |
| `bio.html` | Sections (Bio) |
| `apps.html` | Sections (Apps) |
| `blog.html` | Sections (Blog) |
| `papers.html` | Sections (Publications) |
| `fun.html` | Sections (Fun) |

## Steps

1. After completing changes to any website file, check if the change affects content described in `README.md`
2. If so, update the corresponding section in `README.md`
3. Commit all changes together (both the website file and README updates)
